# Cache

本文档记录和cache有关的所有内容



## memory种类

### Dynamic RAM

俗语“内存”

* 存储空间需要经常刷新

![image-20241023214313329](../statics/image-20241023214313329.png)



### Static RAM

* 不一定需要被"refreshed"
* 比DRAM快得多
* 很贵
